# 时序电路合成完全指南 —— Moore/Mealy 自动机与电路设计

---

## 一、Moore 与 Mealy 自动机

### 1.1 核心区别

| 特性 | Moore 自动机 | Mealy 自动机 |
|------|:-----------:|:-----------:|
| **输出取决于** | **仅当前状态** | 当前状态 **+** 当前输入 |
| **输出标注位置** | 状态圆圈**内部** | 转移箭头**上** |
| **状态数量** | 通常更多 | 通常更少 |
| **输出响应速度** | 慢一个时钟周期 | 立即响应输入变化 |
| **电路稳定性** | 输出更稳定（无毛刺） | 输入变化可能引起输出毛刺 |

### 1.2 图示对比

```
★ Moore 自动机：输出写在状态圆圈内

         input=0               input=0
    ┌─────────┐            ┌─────────┐
    │   S0    │            │   S1    │
    │ Out=0   │───────────▶│ Out=1   │
    └─────────┘            └─────────┘
         ▲                       │
         │     input=1           │  input=1
         └───────────────────────┘


★ Mealy 自动机：输出写在转移箭头上

             0/0                    0/0
    ┌─────────┐                ┌─────────┐
    │   S0    │───────────────▶│   S1    │
    └─────────┘                └─────────┘
         ▲                           │
         │        1/0                │  1/1
         └───────────────────────────┘

    标注格式: input/output
```

### 1.3 什么时候用哪个？

| 场景 | 推荐 |
|------|------|
| 输出需要对输入**即时反应** | Mealy |
| 输出需要与时钟**严格同步**（无毛刺） | Moore |
| **考试/作业**（没有特别说明） | 两种均可，Moore 更安全 |
| 状态数要求最少 | Mealy |

## 二、物理电路图 —— 时序电路的组成要素

### 2.1 组合逻辑 vs 时序逻辑

```
组合逻辑电路：                  时序逻辑电路：
                                
 输入 ──▶ [门电路] ──▶ 输出      输入 ──▶ [组合逻辑] ──▶ 输出
                                              │
         (纯函数，无记忆)                ┌─────┘
                                        ▼
                                   [状态寄存器]
                                        │
                                        └──▶ 反馈回组合逻辑
                                       
                                   (有记忆，输出依赖历史)
```

### 2.2 时序电路标准结构（★必会画）

```
                      ┌──────────────────────┐
     Input ──────────▶│                      │──────────▶ Output
                      │     组合逻辑          │
                      │  (Comb. Logic)        │
                      │                      │
              ┌──────▶│                      │
              │       └──────────┬───────────┘
              │                  │
              │             Next State
              │                  │
              │                  ▼
              │       ┌──────────────────┐
              │       │   状态寄存器       │
              └───────│  (D Flip-Flops)  │◄── Clock
                      └──────────────────┘
                              │
                         Current State
```

### 2.3 D 触发器基础

```
      ┌─────┐
  D ──┤     ├── Q (当前存入的值)
      │ DFF │
CLK ──┤     ├── Q' (Q 的反)
      └─────┘

特性方程: Q(t+1) = D(t)
```

**D 触发器**在时钟上升沿将 D 的值存入 Q，是时序电路存储 1 bit 状态的基本单元。

> **n 个 D 触发器 → 可编码 2^n 个状态**

## 三、★ 例子：判断时序输入中"1"的个数是单数还是双数

### 题意

> 设计一个时序电路，输入为串行比特流（每个时钟周期输入 1 bit），判断**到目前为止输入中 1 的个数**是奇数还是偶数。
> - 若累计 1 的个数为偶数 → 输出 0
> - 若累计 1 的个数为奇数 → 输出 1

---

### ★ 第1步：画自动机（Moore）

**分析**：只有两种状态 —— 当前已收到偶数个 1（Even），或奇数个 1（Odd）。

```
               input=1
    ┌──────────┐        ┌──────────┐
    │  EVEN    │───────▶│   ODD    │
    │  Out=0   │        │  Out=1   │
    └──────────┘◀───────└──────────┘
               input=1
         ▲                   ▲
         │     input=0       │     input=0
         └───────────────────┘
         (状态保持不动的自环，通常省略不画)
```

> **Moore 机**：输出只取决于状态（Even→0, Odd→1），简单直观。

**等价的 Mealy 机**（仅1个状态就够）：

```
             0/0
    ┌──────────┐
    │   S0     │
    └──────────┘
         ▲  │
         │  │  1/1
         └──┘
```

> Mealy 更省状态，但输出有毛刺风险。

---

### ★ 第2步：状态编码

| 状态 | 编码 Q |
|:----:|:------:|
| EVEN |   0    |
| ODD  |   1    |

仅有 2 个状态 → 只需 **1 个 D 触发器**。

---

### ★ 第3步：画真值表（Moore：输出仅由当前状态决定）

| 当前状态 Q(t) | 输入 X | 下一状态 Q(t+1) | 输出 Z |
|:------------:|:------:|:--------------:|:------:|
| 0 (EVEN) | 0 | 0 (EVEN) | 0 |
| 0 (EVEN) | 1 | 1 (ODD) | 0 |
| 1 (ODD) | 0 | 1 (ODD) | 1 |
| 1 (ODD) | 1 | 0 (EVEN) | 1 |

---

### ★ 第4步：推导表达式

#### 下一状态函数（转移函数）D = Q(t+1)

用卡诺图或观察法：

```
    Q(t)
     0   1
X=0| 0 | 1 |
X=1| 1 | 0 |
```

→ **D = Q ⊕ X**

#### 输出函数（Moore 机特有：输出 = f(当前状态)）

```
Z = Q(t)
```

> 在 Moore 机中，Even 状态 Q=0 输出 Z=0；Odd 状态 Q=1 输出 Z=1。输出直接就是状态本身！

---

### ★ 第5步：组合在一起 —— 完整电路图

```
                     ┌──────────┐
     X ──────┬───────┤  XOR     │
             │       └────┬─────┘
             │            │
             │            ▼  D = Q⊕X
             │       ┌─────────┐
             │       │ D     Q ├──────┬────▶ Z = Q
             │       │   DFF   │      │
             └───────┤ CLK     │      │
                     └─────────┘      │
                         ▲            │
                         │            │
                       Clock          │
                                      ▼
                                    Output
```

**完整标注版**：

```
 控制/数据信号说明：
 ┌─────────────────────────────────────────────────┐
 │  X   : 串行输入比特 (每个时钟周期 1 bit)         │
 │  Clock : 系统时钟                                │
 │  Z   : 输出 (0=偶数个1, 1=奇数个1)               │
 │  DFF : 1个D触发器存储状态 (0=EVEN, 1=ODD)        │
 │  XOR : 组合逻辑实现转换函数 D = Q⊕X               │
 └─────────────────────────────────────────────────┘
```

## 四、★ 参考课件案例：检测连续 3 个 1 的电路

### 题意

> 输入为串行比特流，当检测到**连续 3 个 1**时，输出 Z=1；否则 Z=0。使用 Moore 机实现。

例如：输入 `...0 1 1 1 0...` → 在第三个 1 的时钟周期输出 Z=1。

---

### 第1步：画 Moore 自动机

需要记忆"已经连续看到了几个 1"：

```
  状态含义:
  S0: 当前连续 1 的个数 = 0 (或刚看到0)
  S1: 当前连续 1 的个数 = 1
  S2: 当前连续 1 的个数 = 2
  S3: 当前连续 1 的个数 = 3 (检测到目标！)


                1             1             1
         ┌──────┐     ┌──────┐     ┌──────┐     ┌──────┐
         │  S0  │────▶│  S1  │────▶│  S2  │────▶│  S3  │
         │ Out=0│     │ Out=0│     │ Out=0│     │ Out=1│
         └──┬───┘     └──┬───┘     └──┬───┘     └──┬───┘
            │  0         │  0         │  0         │ 0,1
            ▼            ▼            ▼            ▼
           S0           S0           S0           S0
         (回到S0)     (回到S0)     (回到S0)     (回到S0)

简化版（只画关键转移）：
         ┌───┐  1   ┌───┐  1   ┌───┐  1   ┌───┐
    ────▶│S0 │─────▶│S1 │─────▶│S2 │─────▶│S3 │──▶ Z=1
         └─┬─┘      └─┬─┘      └─┬─┘      └─┬─┘
           │0         │0         │0         │0,1
           ▼           ▼          ▼          ▼
          S0          S0         S0         S0
```

> **Moore 特征**：输出 Z 标注在状态圆圈内，S3 的内部写 "Out=1"。

---

### 第2步：状态编码

4 个状态 → 需要 **2 个 D 触发器**（Q1, Q0）

| 状态 | Q1 | Q0 |
|:----:|:--:|:--:|
| S0 | 0 | 0 |
| S1 | 0 | 1 |
| S2 | 1 | 0 |
| S3 | 1 | 1 |

---

### 第3步：真值表

| 当前状态 | Q1 Q0 | 输入 X | 下一状态 | D1 D0 | 输出 Z |
|:--------:|:-----:|:------:|:--------:|:-----:|:------:|
| S0 | 00 | 0 | S0 | 00 | 0 |
| S0 | 00 | 1 | S1 | 01 | 0 |
| S1 | 01 | 0 | S0 | 00 | 0 |
| S1 | 01 | 1 | S2 | 10 | 0 |
| S2 | 10 | 0 | S0 | 00 | 0 |
| S2 | 10 | 1 | S3 | 11 | 0 |
| S3 | 11 | 0 | S0 | 00 | 1 |
| S3 | 11 | 1 | S3 | 11 | 1 |

> **注意**：图中 S3 遇到 1 时保持 S3（因为连续 3+ 个 1 也算"连续 ≥3 个 1"）。
> 如果你的题意是"恰好 3 个"而非"≥3 个"，则 S3→1→S1。这里按课件常见版本保留 S3。

---

### 第4步：推导表达式

#### D1（下一状态的 Q1）

```
    Q1Q0
     00  01  11  10
X=0| 0   0   0   0
X=1| 0   1   1   1
```

SOP 圈法：
- X=1, Q1Q0=01+11+10 → **D1 = X·Q0 + X·Q1**

化简：**D1 = X·(Q1 + Q0)**

#### D0（下一状态的 Q0）

```
    Q1Q0
     00  01  11  10
X=0| 0   0   0   0
X=1| 1   0   1   0
```

SOP 圈法：
- X=1, Q1Q0=00+11 → **D0 = X·Q1'·Q0' + X·Q1·Q0 = X·(Q1⊙Q0)**

或等价地：**D0 = X·(Q1⊕Q0)'**

#### 输出 Z（Moore：仅取决于当前状态）

```
Z = Q1·Q0
```

> 当状态编码为 S3=11 时输出 1；其他状态输出 0。

---

### 第5步：完整电路图

```
                    ┌──────────────────────────────────────┐
                    │           组合逻辑                      │
                    │                                       │
     X ──────┬──────┤                                       │
             │      │  D1 = X·(Q1+Q0)                      │
             │      │               ┌───┐                   │
             │      │  Q1 ────┬────┤OR ├──┐                │
             │      │         │    └───┘  │   ┌───┐        │
             │      │         └───────────┴───┤AND├─▶ D1   │
             │      │                         └───┘         │
             │      │                           ▲           │
             │      │  X ───────────────────────┘           │
             │      │                                       │
             │      │  D0 = X·(Q1⊕Q0)'                     │
             │      │         ┌───┐    ┌───┐               │
             │      │  Q1 ────┤XOR├────┤NOT├──┐            │
             │      │  Q0 ────┤   │    └───┘  │   ┌───┐   │
             │      │         └───┘           └───┤AND├─▶D0│
             │      │                         ┌───┤   │    │
             │      │  X ─────────────────────┘   └───┘    │
             │      │                                       │
             │      │  Z = Q1·Q0                           │
             │      │         ┌───┐                        │
             │      │  Q1 ────┤AND├──▶ Z                  │
             │      │  Q0 ────┤   │                        │
             │      │         └───┘                        │
             └──────┼──────────────────────────────────────┘
                    │
                    ▼
              ┌──────────┐      ┌──────────┐
              │ D    Q   │      │ D    Q   │
              │   DFF1   │      │   DFF0   │
              │ CLK      │      │ CLK      │
              └────┬─────┘      └────┬─────┘
                   │  Q1              │  Q0
                   └──┬───────────────┘
                      │
                      ▼
                 反馈到组合逻辑
                      ▲
                      │
                    Clock
```

**简化符号版（考试答题推荐）**：

```
                         ┌──────────────────────┐
        X ──────────────▶│                      │──────▶ Z
                         │    组合逻辑            │
        Q1 ◀─────────────│                      │──────▶ D1
        Q0 ◀─────────────│                      │──────▶ D0
                         └──────────────────────┘
                                   │
                              D1   │   D0
                                   ▼
                         ┌──────────────────┐
                         │  Q1(D1)   Q0(D0) │
                         │    2个 D触发器     │◄── CLK
                         └──────────────────┘
```

## 五、时序电路合成 5 步法总结

```
┌─────────────────────────────────────────────────────────────┐
│ 第1步：画自动机（Moore 或 Mealy）                              │
│   → 明确：几个状态？每个状态什么含义？什么条件下转移？           │
├─────────────────────────────────────────────────────────────┤
│ 第2步：状态编码                                               │
│   → n 个状态 → ⌈log₂n⌉ 个 D 触发器                           │
│   → 给每个状态分配唯一的二进制编码                              │
├─────────────────────────────────────────────────────────────┤
│ 第3步：画真值表                                               │
│   → 列：当前状态(Q)、输入(X)、下一状态(Q+)、输出(Z)              │
│   → 输入行数 = 状态数 × 2^(输入位数)                           │
├─────────────────────────────────────────────────────────────┤
│ 第4步：推导表达式 + 电路图                                     │
│   → 转移函数 Di = f(Q, X)   （每个触发器的 D 输入）             │
│   → 输出函数 Z = g(Q) [Moore] 或 g(Q, X) [Mealy]             │
│   → 用卡诺图化简 → 画出组合逻辑部分                             │
├─────────────────────────────────────────────────────────────┤
│ 第5步：组合在一起                                              │
│   → 组合逻辑 + D 触发器 + 反馈回路 + 时钟                       │
│   → 画出完整的时序电路图                                       │
└─────────────────────────────────────────────────────────────┘
```

## 六、⚠️ 重难点与易错提醒

| 重难点 | 解析 |
|--------|------|
| **Moore vs Mealy 输出时机** | Moore 输出变化发生在**时钟沿后**（状态更新后）；Mealy 输出在**输入变化时立即**变化 |
| **状态编码不是唯一的** | 不同编码导致不同电路复杂度。格雷码编码通常能简化卡诺图 |
| **自环要不要画** | Moore 机中状态保持不动的转移（输入 0 时 Even→Even）可省略，但真值表必须写全 |
| **未用状态的处理** | 如果状态数不是 2 的幂，未用的编码应在真值表中设为"无关项(don't care)"以简化电路 |
| **D 触发器数量** | 2 状态→1 个、3-4 状态→2 个、5-8 状态→3 个，依此类推 |
| **输出函数≠转移函数** | Moore 机中输出 Z 仅依赖当前 Q，不依赖 X —— 这就是跟 Mealy 的本质区别 |
| **异步复位** | 实际电路中通常加 Reset 信号让电路从 S0 开始工作 |
